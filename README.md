# User Persona Development

## Internship Project

**Intern ID:** CITS3057

### Project Overview

This repository contains the User Persona Development project completed as part of the UX Design Internship. The objective of this project is to create a detailed user persona that represents the target audience for a movie recommendation application.

### Repository Contents

* **README.md** – Project overview and internship details
* **Report.pdf** – Detailed User Persona report
* **User_Persona.png** – Designed user persona

### Objective

* Understand the target user.
* Identify user goals, behaviors, needs, and frustrations.
* Support user-centered design decisions.
* Improve the overall user experience of the application.

### Tools Used

* Canva / Figma (for persona design)
* Microsoft Word (for report preparation)
* GitHub (for project repository)

### Learning Outcomes

* Learned the importance of user personas in UX design.
* Understood how user research supports product design.
* Improved documentation and GitHub repository management skills.
